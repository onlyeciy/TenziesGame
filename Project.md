Tenzies Game Project

How this game work?
There will be 10 dice with random value, our job is to make all of that 10 dice has the same value

In this project we are able to:

1.Click each die to lock it's value
2.roll button if there's no other die with the same value left. it will randomize the remaining dice
3.if all the dice has the same value, means we win the game and be able to restart
